Users manual
============

Usage
-----

Create a new field for your form. Set the marker equal to the equivalent salesforce field.

.. figure:: /Images/AdministratorManual/CustomMarker.png
  :width: 500px
  :alt: Set custom marker

**Note:** Because salesforce uses case-sensitive Variables, powermails lower-case conversion is deactivated by default.

FAQ
^^^

| **Q: Does this work with double opt in?**
| A: Yes, this extension adds an powermail finisher and will respect the powermail workflow.

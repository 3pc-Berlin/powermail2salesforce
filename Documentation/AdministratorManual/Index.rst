Administrator Manual
====================


Installation
------------

To add salesforce support do following:

1. If not already created, create a new extension template on your form page.
2. Include the **Powermail - Salesforce** Template.
3. Go into the constants editor and set the constants for the **PLUGIN.TX_POWERMAIL_SALESFORCE** Category.
